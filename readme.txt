## Project Overview
This project is a **minimal yet functional in-memory cache database**, inspired by Redis. It supports multiple data types (strings, lists, sets, hash maps) and provides **fast storage/retrieval, TTL expiration, and network access via a TCP server**.  

**Repository:** [GitHub Link](https://github.com/murtaza335/in-memory-cache-db-cpp)
